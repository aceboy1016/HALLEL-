# HALLEL GASスクリプト

各店舗のGoogle Apps Scriptを管理するディレクトリです。

## ディレクトリ構成

```
gas-scripts/
├── shibuya/          # 渋谷店（APIのみ）
├── nakameguro/       # 中目黒店（APIのみ）
├── yoyogiuehara/     # 代々木上原店（APIのみ）
├── ebisu/            # 恵比寿店（API + カレンダー同期）
└── hanzomon/         # 半蔵門店（API + カレンダー同期）
```

## 各店舗の設定

| 店舗 | カレンダー同期 | 部屋/エリア |
|------|--------------|------------|
| 渋谷 | なし | STUDIO 1~7 |
| 中目黒 | なし | フリーウエイトエリア / 格闘技エリア |
| 代々木上原 | なし | なし（2枠区別なし） |
| 恵比寿 | あり | 個室A / 個室B |
| 半蔵門 | あり | STUDIO B 1~3 / 個室A / 個室B / 貸切 |

## 使い方

1. 該当店舗のディレクトリにある `main.js` をコピー
2. GASエディタに貼り付け
3. `setupTrigger10min()` を実行してトリガー設定
